ColorZilla

Une pipette est mise à disposition pour pouvoir prendre les références colorimétriques d'un site. Et ce, que ce soit en héxadécimal ou RGB.

Colorzilla

Font Face Ninja

Lorsque l'on va sur différents sites, certaines typographies peuvent nous sembler intéressantes. Font Face Ninja permet de les mettre en favori, de les essayer et les acheter.

Font Face Ninja

CSS Peeper

Extension très intéressante dans la mesure où il est possible d'analyser une page et de savoir quelle typographie est utilisée, la couleur et les ressources (images par exemple) utilisées sur le site.

CSS Peeper

Windows Resizer

Grâce à cet outil, il est possible d'avoir un aperçu des sites voire le votre sur différents types d'écrans et donc leur résolution.
